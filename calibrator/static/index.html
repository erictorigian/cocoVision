🐹 Coco Cam — Zone Calibrator

Draw polygons on the camera frame to define Coco's activity zones.

How to use:
1. Select a zone from the list above.
2. Click on the image to place polygon points.
3. Double-click (or click near first point) to close the polygon.
4. Repeat for each zone.
5. Click Save Zones when done.

Re-run anytime you rearrange the cage.
